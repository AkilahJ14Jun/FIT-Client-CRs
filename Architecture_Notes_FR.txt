  Key architectural relationships:
  1. Frontend → Backend: All data requests go through the /api proxy (localhost:3001)
  2. Backend → Database: TypeORM ORM handles database interactions
  3. Backend → WhatsApp: PowerShell scripts handle OS-level protocol
  4. Database → Frontend: Data is returned through API responses
  5. Frontend → WhatsApp: PDF receipts are generated and shared via browser
